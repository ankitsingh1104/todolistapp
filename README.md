A simple, intuitive, and efficient To-Do List Android application designed to help users organize daily tasks, improve productivity, and manage time effectively. The app allows users to create, update, mark, and delete tasks with a clean and user-friendly interface.

# Features
Add new tasks with title and details
Edit existing tasks
Mark tasks as completed / uncompleted
Delete tasks easily
View total task count
Color-coded tasks for better visual organization
Local data storage for offline usage

# Tech Stack

Language: Kotlin
Architecture: MVVM (Model–View–ViewModel)
UI Design: XML + Material Design Components
Database: Room (SQLite)
State Management: LiveData
Dependency Injection: ViewModelProvider
Build Tool: Gradle

# App Architecture

The application follows the MVVM architecture, ensuring:
Separation of concerns
Better maintainability
Easy testing and scalability

# Layers:

Model: Handles data (Room Database, DAO, Entities)
View: Activities/Fragments and XML layouts
ViewModel: Business logic and LiveData

# User Interface

Minimal and clean layout
Easy navigation
Responsive design for different screen sizes
Material Design principles for better UX
